# 카드 혜택매니저 PWA 업데이트 현황

## 현재 개발 기준

**현재 최신 테스트 버전:** `v7.0.0 UI Beta`  
**기준 Stable:** `v6.5 Stable`  
**프로젝트:** 카드 혜택매니저 PWA  
**업데이트 기준일:** 2026-07-03

---

## 핵심 개발 원칙

- Stable 버전은 직접 수정하지 않는다.
- 새 기능은 Beta에서 먼저 테스트한다.
- 정상 확인 후 Stable로 승격한다.
- 데이터 구조와 백업 호환성을 최우선으로 유지한다.
- UI 변경 시에도 카드 데이터, 히스토리, GitHub 백업 구조는 건드리지 않는다.

- 백업 구조 변경 없음

- 모바일, iPad, PC를 각각 다른 화면 구조로 최적화

#### 주요 변경

## 설치 및 확인 메모

GitHub Pages 배포 후 캐시가 남아 있으면 주소 뒤에 버전 파라미터를 붙여 확인한다.

예시:

```text
https://justin-1984.github.io/card-Benefit-manager/?v=700
```

버전 확인 후 문제가 있으면 이전 Stable 또는 Beta ZIP으로 되돌린다.
