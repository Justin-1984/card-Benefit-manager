# CHANGELOG



## v7.4.6 Full Subscription Burden
- 카드사 현재 사용금액과 예정 구독을 분리
- 실적 입력 시 금액 확인 기준일 저장
- 기준일 이후 결제 예정 구독만 예상 실적에 포함
- 구독 금액을 현재 사용금액에 직접 더하지 않도록 변경
- 카드별 현재 사용금액/예정 구독/예상 실적/추가 사용 필요액 표시

## v7.4.4 Subscription Calendar & UI Refine
- 구독 캘린더를 월간 캘린더형으로 개선
- 카드 추천은 실제 변경이 필요한 구독만 표시
- 구독 분석을 요약형으로 정리
- 구독 리스트를 접기/펼치기 가능하게 변경

## v7.4.3 Settings Nav Restore
- 모바일 하단 탭에서 설정 버튼이 사라져 보이던 문제 수정
- 기존 모바일 CSS의 2열 탭바 강제값을 3열 탭바로 재정의
- 상단 헤더에 설정 바로가기 버튼 추가
- 데이터 구조 변경 없음

## v7.4.2 Card Recommendation Apply
