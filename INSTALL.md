# 설치 및 업데이트 방법

## GitHub Pages 배포

1. ZIP 파일을 압축 해제합니다.
2. 저장소 루트에 파일을 업로드합니다.
3. GitHub Pages가 활성화되어 있는지 확인합니다.
4. 브라우저에서 Pages 주소로 접속합니다.

## 강제 업데이트 확인

캐시가 남아 있으면 주소 뒤에 버전 파라미터를 붙입니다.

```text
https://justin-1984.github.io/card-Benefit-manager/?v=710
```

## iPhone/iPad 설치

1. Safari에서 GitHub Pages 주소 접속
2. 공유 버튼 선택
3. 홈 화면에 추가
4. 앱처럼 실행

## 업데이트 전 주의

- 기존 Stable 백업을 먼저 확보합니다.
- GitHub Backup / JSON Backup을 확인합니다.
- Beta 적용 후 데이터가 정상인지 확인합니다.
- 문제가 있으면 이전 Stable 또는 이전 Beta ZIP으로 되돌립니다.
